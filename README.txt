SparkCentral Project
By Matt Ward

This project contains 2 main classes.

1.	Calculator - this stores the operators and equation to be processed, new operators
	can be added to default operators, order of operation is as defined in operators array.
2.	Operator - Provides calculation for a specific native PHP operator on 2 numbers, native 
	PHP operators can be assigned any arbitrary string identifier.

An example program can be executed by running the main.php file included.
